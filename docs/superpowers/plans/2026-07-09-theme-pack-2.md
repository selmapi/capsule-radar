## Task 2: Blip-shape dispatch + draw functions + outline (the rendering task)

**Files:** `src/radar_view.cpp` (`ac_draw_cb` + new static draw helpers)

- [ ] **Step 1 — New static helpers** near `draw_bracket` (all take `lv_draw_ctx_t *d, const AcDraw &ac`; color = `ac.color`; honor `s_desc->outline` by adding a black border to the polygon's `lv_draw_rect_dsc` when true):
  - `draw_chevron` — a 3-point arrowhead pointing along `ac.track`. Reuse the kite's rotation math (the `cos/sin` of `th = ac.track*π/180` used for the GX/GY quad). Local points e.g. `{(0,-11),(9,9),(-9,9)}` rotated + offset to `ac.pos`, drawn with `lv_draw_polygon`.
  - `draw_silhouette` — a ~10-point aircraft polygon (fuselage nose→tail, swept wings, tailplane), rotated by `ac.track`, scale ~13px. Keep the point list readable and commented. `lv_draw_polygon`.
  - `draw_diamond` — an axis-aligned 4-point diamond around `ac.pos` (NOT rotated), half-size ~9px: points `{(0,-9),(9,0),(0,9),(-9,0)}`. `lv_draw_polygon`.
  - For all three, if `s_desc->outline`, set `dsc.border_color = lv_color_black(); dsc.border_width = 2; dsc.border_opa = LV_OPA_COVER;` on the rect dsc so the fill gets a black cel-shade edge.

- [ ] **Step 2 — Dispatch in `ac_draw_cb`.** In the non-orb (`else`) branch, after `draw_trail(...)`, replace the current `if (vec) draw_bracket(...); else { kite quad }` with a shape resolve + switch:
```cpp

    if (sh == BlipShape::kAuto) {           // resolve to the scope's natural marker
        if (vec) { draw_bracket(d, ac); }
        else     { /* existing rotated kite quad code, unchanged */ }
    } else if (sh == BlipShape::kChevron)    { draw_chevron(d, ac); }
      else if (sh == BlipShape::kSilhouette) { draw_silhouette(d, ac); }
      else if (sh == BlipShape::kDiamond)    { draw_diamond(d, ac); }
```
Keep the emergency halo, selection ring, and floating labels AFTER this block, unchanged, applying to all shapes. Namespace-qualify `BlipShape` as needed (`radar::BlipShape`) matching how `s_desc`/`radar::ScopeStyle` are referenced in this file.
